## 이번 세션 요약

### 1. 번개탭 마커/목록 이미지 불일치 근본 수정 ✅
- **근본 원인 1**: `naverMapService.ts`에서 데모 사진을 배열 인덱스(`idx`) 기반으로 할당 → 서버 응답 순서 변경 시 동일 유저에 다른 사진
- **수정**: `stablePhotoIndex(user_id)` 해시 함수 도입 → 동일 user_id = 동일 사진 보장
- **근본 원인 2**: `Image.prefetch()` 비동기 게이트로 `mapMarkerUsers` 설정 지연 → 목록과 마커 간 데이터 불일치
- **수정**: prefetch 제거, 동기 설정
- **근본 원인 3**: `expo-file-system` 네이티브 모듈 미링크 → `FileSystem.downloadAsync` 전부 실패 (마커 항상 fallback 표시)
- **수정**: `pod install` → ExpoFileSystem 네이티브 모듈 빌드 포함 확인

### 2. 원형 마커 이미지 처리 ✅

- `demoMarkerImages.ts`: URL → 번들 원형 이미지 O(1) 매핑
- 실제 이미지: `FileSystem.downloadAsync` → 로컬 캐시 경로 사용
