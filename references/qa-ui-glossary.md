# ZZIRIT QA 용어 사전

QA 이슈 등록 시 사용하는 공통 용어입니다.
**재현 경로와 오류 설명에 아래 용어를 사용해주세요.**

---

## 1. 화면(Surface) 이름

| 화면 | 설명 | 진입 경로 |
|------|------|-----------|
| **login** | 로그인/회원가입 | 앱 최초 실행 |
| **onboarding** | 회원가입 후 프로필 설정 플로우 | 회원가입 완료 후 자동 진입 |
| **lightning** | 번개탭 — 지도 기반 주변 유저 탐색 | 하단 탭 바 > 번개 |
| **meeting** | 미팅탭 — 그룹 미팅 목록/생성 | 하단 탭 바 > 미팅 |
| **chat** | 채팅탭 — 1:1 대화 목록/채팅방 | 하단 탭 바 > 채팅 |
| **my** | MY탭 — 내 프로필/설정 | 하단 탭 바 > MY |
| **likes** | 좋아요 — 받은/보낸/매칭 목록 | MY탭 > 좋아요 영역 |
| **billing** | 볼트 충전/결제 | MY탭 > 볼트 충전 |

---

## 2. 내비게이션 구조

### 하단 탭 바
앱 하단에 항상 표시되는 4개 탭입니다.

```
[ 번개 ] [ 미팅 ] [ 채팅 ] [ MY ]
```

- **번개**: 지도 아이콘
- **미팅**: 사람들 아이콘
- **채팅**: 말풍선 아이콘
- **MY**: 내 프로필 사진 (원형)

### 화면 이동 방식

| 용어 | 설명 |
|------|------|
| **탭 전환** | 하단 탭 바에서 다른 탭을 누름 |
| **화면 진입** (push) | 현재 화면 위에 새 화면이 쌓임. 좌상단 ← 뒤로가기로 복귀 |
| **모달** | 화면 위에 떠오르는 팝업 창. 배경이 어두워짐 |
| **바텀시트** | 화면 하단에서 올라오는 패널. 위아래로 드래그하여 높이 조절 가능 |
| **다이얼로그** | 화면 중앙에 나타나는 확인/취소 창 |
| **토스트** | 화면 상단/하단에 잠깐 나타났다 사라지는 알림 메시지 |

## 3. 공통 UI 요소

### 버튼

| 용어 | 설명 | 예시 |
|------|------|------|
| **CTA 버튼** | 화면의 주요 행동 버튼. 보통 크고 색이 채워져 있음 | "시작하기", "충전하기" |
| **텍스트 버튼** | 배경 없이 텍스트만 있는 버튼 | "건너뛰기", "자세히 보기" |
| **아이콘 버튼** | 아이콘만 있는 버튼 | ← 뒤로가기, × 닫기, ⋯ 더보기 |
| **FAB** | 화면 위에 떠 있는 원형 버튼 (Floating Action Button) | 지도 위 내 위치 버튼 |

### 입력

| 용어 | 설명 | 예시 |
|------|------|------|
| **텍스트 입력** | 키보드로 텍스트를 입력하는 필드 | 닉네임, 자기소개 |
| **드롭다운** | 탭하면 선택지 목록이 펼쳐지는 필드 | 직업, 학력 선택 |
| **칩 (Chip)** | 탭하여 선택/해제하는 작은 태그 형태 | 관심사 선택, 키워드 |
| **토글** | 켜기/끄기 스위치 | 알림 설정 on/off |
| **라디오** | 여러 개 중 하나만 선택 가능한 원형 버튼 | 성별 선택 |
| **체크박스** | 여러 개를 동시에 선택 가능한 사각형 버튼 | 약관 동의 |

### 표시

| 용어 | 설명 | 예시 |
|------|------|------|
| **아바타** | 원형 프로필 사진 | 채팅 목록의 상대방 사진 |
| **카드** | 정보를 묶어서 보여주는 사각형 영역 | 미팅 카드, 프로필 카드 |
| **배지** | 아이콘/아바타 위에 붙는 작은 숫자/점 | 안 읽은 채팅 수 |
| **블러** | 콘텐츠를 흐릿하게 가려놓은 상태 | 좋아요 받은 프로필 (잠금) |
| **오버레이** | 화면 위에 반투명하게 덮인 영역 | 프로필 카드 위 키워드 칩 |
| **빈 상태** | 데이터가 없을 때 보여주는 안내 화면 | "아직 채팅이 없습니다" |
| **로딩** | 데이터를 불러오는 중 표시 | 스피너, 스켈레톤 |

## 4. 화면별 주요 요소

### lightning (번개탭)

| 요소 | 설명 |
|------|------|
| **지도** | 네이버 지도. 드래그로 이동, 핀치로 확대/축소 |
| **프로필 마커** | 지도 위 원형 프로필 사진 (노란 테두리) |
| **번개 버블** | 지도 위 이모지 마커 (🍕🍻🎬☕️). 번개 미팅 위치 표시 |
| **내 위치 버튼** | 지도 우하단 FAB. 탭하면 내 위치로 이동 |
| **번개 목록 시트** | 하단에서 올라오는 바텀시트. 주변 유저 리스트 |
| **프로필 카드** | 유저 상세 정보 카드. 사진 + 이름/나이/직업/거리 + 키워드 칩 |
| **액션 버튼** | 프로필 카드 하단: ✕(넘기기), 💛(좋아요), 💬(채팅) |
| **볼트 차감 다이얼로그** | "좋아요를 누를까요?" (10볼트) / "채팅방을 만들까요?" (15볼트) |

### meeting (미팅탭)

| 요소 | 설명 |
|------|------|
| **필터 버튼** | 미팅 목록 상단. 조건별 필터링 |
| **미팅 카드** | 미팅 정보 카드: 작성자, 제목, 위치, 인원, 상태 |
| **모집마감 배지** | 미팅이 마감된 상태 표시 |
| **미팅 생성 폼** | 새 미팅 만들기 화면. 제목/위치/인원/설명 입력 |

### chat (채팅탭)

| 요소 | 설명 |
|------|------|
| **채팅 목록** | 대화 상대 리스트. 아바타 + 이름 + 마지막 메시지 + 시간 |
| **안읽음 배지** | 채팅 목록에서 안 읽은 메시지 수 표시 |
| **채팅방** | 1:1 대화 화면 |
| **메시지 버블** | 대화 말풍선. 내 것은 우측(초록), 상대는 좌측 |
| **입력바** | 채팅방 하단. 텍스트 입력 + 전송 버튼 |
| **첨부 메뉴** | + 버튼 탭 시 나오는 세로 목록: 카메라 / 사진 / 위치 |
| **위치 메시지** | 지도 프리뷰 이미지 + "위치 확인하기 >" |
| **사진 메시지** | 전송된 이미지. 탭하면 전체 화면 |

### my (MY탭)

| 요소 | 설명 |
|------|------|
| **프로필 요약** | 상단 영역. 프로필 사진 + 이름 + 기본 정보 |
| **프로필 편집** | 15개 필드 편집 화면 (닉네임, 직업, 키, 체형, MBTI 등) |
| **사진 관리** | 최대 6장, 드래그로 순서 변경 |
| **사진 확인 모달** | 사진 선택 후 확인/취소 팝업 |
| **설정** | 계정관리, 앱정보, 약관, 정책 |
| **볼트 영역** | 현재 볼트 잔액 + 충전하기 진입 |

### likes (좋아요)

| 요소 | 설명 |
|------|------|
| **탭 (상단)** | 3개 탭: 받은 좋아요 / 보낸 좋아요 / 찌릿 |
| **블러 카드** | 받은 좋아요에서 잠긴 프로필 (흐릿하게 표시) |
| **잠금해제 다이얼로그** | "프로필을 확인할까요?" + 볼트 차감 안내 |
| **프리뷰 모달** | 잠금해제 후 프로필 미리보기 |

## 5. 재현 경로 작성법

### 기본 형식

```
{탭} > {화면} > {요소} > {동작}
```

### 예시

```
번개탭 > 프로필 카드 > 💛 좋아요 버튼 탭
→ 볼트 차감 다이얼로그 표시됨
→ "확인" 탭
→ 기대: 좋아요 전송 + 10볼트 차감
→ 실제: 아무 반응 없음
```

```
채팅탭 > 채팅 목록 > 첫번째 대화방 탭 > 채팅방 진입
→ 첨부(+) 버튼 탭
→ 첨부 메뉴 > "카메라" 탭
→ 기대: 카메라 실행
→ 실제: 에러 토스트 표시
```

```
MY탭 > 프로필 편집 > 관심사 칩 선택
→ 3개 선택 후 "저장" 탭
→ 기대: 선택한 관심사 저장됨
→ 실제: 저장 후 다시 들어가면 초기화됨
```

## 6. 심각도 기준

| 등급 | 기준 | 예시 |
|------|------|------|
| **critical** | 앱 크래시, 데이터 유실 | 결제 중 앱 종료, 채팅 메시지 소실 |
| **major** | 핵심 기능 동작 불가 | 로그인 안됨, 카메라 에러, 위치 전송 실패 |
| **minor** | 기능은 되지만 불편 | 버튼 반응 느림, 스크롤 끊김 |
| **cosmetic** | 디자인/텍스트 차이 | 간격 다름, 색상 불일치, 오타 |
