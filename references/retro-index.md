# 회고 인덱스 (Retro Index)

> 모든 세션 회고와 교훈을 한곳에서 조회.
> **새 세션 시작 시 반드시 이 문서를 먼저 읽는다.**

## 핵심 교훈 (반복 실수 방지)

### 코드 변경
- **한 커밋 = 한 목적** — 관련 없는 변경을 같은 커밋에 포함하지 않는다 (s7)
- **삭제 전 사용처 확인** — `grep -r "ComponentName"` 으로 import 확인 후 삭제 (s7)
- **변경 후 전체 플로우 테스트** — 파일 수정 시 해당 화면의 전체 사용자 플로우 시뮬레이터 확인 (s7)

### 데이터
- **실제 유저 데이터 절대 덮어쓰기 금지** — 시드 스크립트가 메인 유저 프로필을 수정하면 안 됨 (s7)
- **필드 네이밍 이중화 주의** — 앱 Firestore 직접읽기 vs 서버 API 필드명 불일치 ([[field-naming-map]]) (s7)

## 리그레션 체크리스트

큰 변경(10+ 파일) 후 반드시 확인:

- [ ] **번개탭**: 지도 로딩 → 마커 표시 → 마커 클릭 → 프로필 카드 피드 드래깅 → 상세 진입 → 찌릿/좋아요 버튼
- [ ] **좋아요**: 받은/보낸/ZZIRIT 탭 전환 → 카드 표시 → 블러/결제
- [ ] **채팅**: 채팅 목록 로딩 → 채팅방 진입 → 메시지 표시 (텍스트/이미지/위치)
- [ ] **MY**: 프로필 이미지 (본인 업로드) → 이름 일치 → 볼트 잔액 → Like 섹션
- [ ] **미팅**: 목록 로딩 → 상세 → 작성 → 작성자 이름/이미지 일치
