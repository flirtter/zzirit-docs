# Likes Surface Spec

## Like 화면 (MY탭 하위)
- 받은 Like, 보낸 Like, ZZIRIT 3개 탭

### 받은 Like
- 탭명: "받은 Like {개수}"
- **라이크보기 구매자**: 프로필 썸네일 정상 노출, 이름/나이/직업, 클릭→카드
- **미구매자**: 블러/블라인드, 이름/나이/직업 최대 6개, 클릭 불가
- 하단 "내가 받은 라이크 확인하기" 버튼 (미구매자에게만)
  - 볼트 있는 경우: "볼트를 사용할까요?" 팝업 → 취소/사용하기
  - 볼트 부족: "볼트가 부족해요" 팝업 → 취소/충전하기 (150볼트 추천)

### 보낸 Like
- 탭명: "보낸 Like" (개수 미표시)
- 모든 유저에게 정상 노출, 이름/나이/직업, 클릭→카드

## 좋아요 과금
- 1일 20회 무료
- 초과 시: 좋아요 1회 → 10볼트
- 무료 횟수 차감 팝업: "좋아요를 누를까요? 10볼트가 차감돼요."
- 매칭(서로 like): 무료 채팅방 생성

## 받은 Like 보기 과금
- 무료 서비스 X
- 1회 보기: 15볼트
- 1시간 무제한: 120볼트
