# Billing (볼트/과금) Surface Spec

## 재화
- 이름: 볼트
- 단위: 1볼트 = 100원

## 구매 단위
| 볼트 | 가격 |
|------|------|
| 40 | 4,000원 |
| 80 | 8,000원 |
| 100 | 10,000원 |
| 150 | 15,000원 |
| 200 | 20,000원 |
| 250 | 25,000원 |

## 무료 요소
- 좋아요: 1일 20회 무료
- 채팅방 만들기: 1일 5회 무료 (익명+공개 합산)
- 서로 찌릿 매칭: 항상 무료 채팅방
- 채팅 메시지: 무제한
- 미팅 게시글 올리기: 항상 무료

## 과금 요소
- 좋아요 20회 초과: 1회 → 10볼트
- 채팅방 만들기 5회 초과: 1회 → 15볼트
- 받은 좋아요 보기: 1회 15볼트 / 1시간 무제한 120볼트

## 무료 볼트 지급
- 최초 가입: 40볼트 (오픈 이벤트 시 80볼트)
- 프로필 자기소개 모두 입력: 10볼트

## 결제 방식
- 앱스토어 인앱결제만 지원

## 팝업 종류
### 무료 횟수 차감 팝업
- 채팅방: "무료 채팅 기회 1회가 차감돼요. 남은: N회" → 취소/만들기
- 좋아요: "좋아요를 누를까요? 10볼트가 차감돼요." → 취소/누르기

### 볼트 차감 팝업
- 채팅방: "15볼트가 차감돼요." → 취소/만들기
- 받은 라이크: "볼트 N개가 사용돼요." → 취소/사용하기

### 볼트 부족 팝업
- "볼트가 부족해요. 지금 볼트를 충전하세요." → 취소/충전하기
