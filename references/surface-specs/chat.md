# Chat (채팅) Surface Spec

## 1. 채팅탭
- 1:1 채팅만 제공 (단체 채팅 추후 검토)

### 1-1. 대화 목록
- 참여 중인 채팅방 목록, 최근 메시지 순 정렬
- 안 읽은 메시지: 우측 버블(개수)

## 2. 채팅방
- 하단 4탭 미제공

### 2-a. 대화 상대 프로필
- **메시지 없을 때**: 상단 크게 - 프사, 이름, 학교/회사, 상태메시지
- **메시지 있을 때**: 최상단 축약 - 프사, 이름, 학교/회사

### 2-b. 프로필 바로가기 버튼
- 프로필 하단 → 상세 이동

### 2-c. 예시 메시지 버블 (5개)
- 안녕하세요! / 같이 노실래요? / 반가워요!👋 / 지금 뭐해요? / 친해지고 싶어요!
- 탭 → 입력창에 입력

### 2-d. + 버튼
- 카메라: 바로 촬영
- 사진 보내기: 앨범에서 1장 선택
- 내 위치 보내기: 지도에서 위치 선택

### 2-e. 메시지 입력창 + 전송 버튼

### 2-f. 수신 메시지
- 프사, 이름, 내용, 시간
- 롱탭 → 신고하기

### 2-g. 발신 메시지
- 내용, 시간, 읽음 표시

### 2-h. 위치 첨부 메시지
- 위치 아이콘 + 위치명 + 밑줄

### 2-i. 사진 첨부 메시지
- 3:4 비율 크롭

## 3. 찌릿 매칭 채팅방
- 서로 like → 자동 생성
- 별도 스킨 노출
- "⚡찌릿 매칭 성공! 이제 말 걸어도 되는 사이 😏"
