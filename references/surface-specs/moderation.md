# Moderation (신고/징계) Surface Spec

## 신고
### 진입점
- 타 유저 프로필 카드
- 타 유저 채팅 메시지 (롱탭)
- 타 유저 미팅 게시판 글

### 신고 사유
- 음란성/성적 발언
- 욕설/혐오 표현
- 사기/금전 요구
- 광고/홍보 목적
- 허위 정보/사진 도용
- 기타

## 징계 정책
### 주의
- 팝업: "서비스 운영 정책에 부합하지 않는 부분이 확인되었습니다..."
- 버튼: 확인 / 운영 정책 보기
- 처리: 별도 제한 없음

### 이용 정지 (7일/14일/30일)
- 팝업: "N일 간 서비스 이용이 제한되었습니다..."
- 버튼: 확인 / 운영 정책 보기
- 정지 중:
  - 번개탭: 프로필 미노출
  - 채팅: 블라인드, 채팅방 진입 불가, "현재 이 회원과의 채팅을 이용할 수 없어요"
  - 미팅: 메시지 보내기 불가
- 복귀 시: 프로필 초기화, 미팅글/채팅방 삭제, 온보딩 프로필 설정 재노출

### 계정 삭제
- 서비스 내 모든 프로필/콘텐츠 삭제
- 이메일로 계정 삭제 안내 발송
- 로그아웃 처리

## 푸시 알림
### 채팅 메시지
- 조건: 새 채팅 메시지 수신
- 대상: 모든 유저 (신고 유저 제외)
- 랜딩: 해당 채팅방

### 찌릿 매칭 성공
- 조건: 서로 like로 채팅방 생성
- 대상: 모든 유저
- 랜딩: 해당 채팅방

### 좋아요 수신
- 조건: 2명 이상 좋아요
- 대상: 라이크 유료 구독 유저
- 예외: 신고 유저 제외
